# Weather App

A simple and clean Weather Application that shows real-time weather data for any city.

### Live Demo
[View Live Demo](https://zestymec.github.io/weatherapp/)

### Features
- Search weather by city name
- Shows current temperature in Celsius
- Displays humidity and wind speed
- Dynamic weather icon
- Fully responsive design

### Technologies Used
- HTML5
- CSS3
- JavaScript (Async/Await + Fetch API)
- OpenWeatherMap API

### How to Use
1. Enter any city name in the search box
2. Click the search button or press Enter
3. Get instant weather information

### Note
This app uses a free public API. No personal data is collected or stored.
